# Cách khởi chạy:

- Với server: 
```bash 
cd server
npm install
node index.js
```
Nếu debug có thể dùng nodemon

- Với client: 
```bash 
cd client
npm install
npm run dev
```

- Mở 2 tab hoặc hơn cho client (thường là `http://localhost:5173/`), tab đầu tiên sẽ cầm token, nhấn pass token để truyền sang tab thứ 2, ...
